DotA 2 Team Composition Statistics

These statistics were created using an open source C# application which parses data from Dotabuff. A total of team compositions were evaluated according to their win ratios. Not all of the compositions were included in these statistics, as only few met the requirement of at least 50 samples each. The flags/roles you can see in the tables, such as "carry" and "disabler", were simply copied from the game itself. They represent Valve's categorisation of the heroes and do not necessarily do them justice. The width of the bars was normalised according to the highest win ratio across all categories.

Some of the configurations might add up to more then 5 flags in total. This is not because the system is broken but because Valve use 3-4 different flags for the same hero so they will be counted in multiple categories simultaneously.

Feel free to leave comments about these statistics on Reddit. Thanks for all your feedback, by the way! I have already added numerous suggestions to the to-do list of the project.
